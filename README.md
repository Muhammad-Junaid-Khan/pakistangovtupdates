# PakistanGovtUpdates.com

Static informational portal for Pakistani government jobs, schemes, NAVTTC courses, scholarships, and application guidance.

## Setup
- Replace EmailJS placeholders in `assets/js/main.js` with your EmailJS `USER_ID`, `SERVICE_ID`, and `TEMPLATE_ID`.
- Deploy on GitHub Pages from the repository root.

## Notes
- The site uses JSON files in `data/` for dynamic content.
- Use `search.html?q=keyword` for client-side search.
